Front-End web development is all about the aesthetics and interactivity of the website. What is important to learn for someone who is looking to get into design/development is first and foremost: HTML. A web developer of any kind should have a good understanding of some version of HTML, weather it be HTML5 or XHTML. HTML allows you to add links, images, paragraphs, and more to your page. Then, you must understand CSS, which stands for Cascading Style Sheet. CSS will allow you to apply style(s) to your web page; in other words, CSS allows you to make your page look asthetically pleasing. Then, one must learn Javascript.
