Building your first Java Program!

This is a tutorial for complete novices. What we're going to do is build a simple program that prints out, "Hello World."

  1. First we need to create our hello world class by writing "public class HelloWorld {" at the top of the page
  2. Then, we must write "public static void main(String[] args) {" to declare our main method.
  3. Within our main method we're going to have to put "System.out.println("Hello World!");" This statement will print out "Hello World!"
  4. Fianlly, put two curly brackets('}') at the end of the program to signify the end of the main method and the end of the class
